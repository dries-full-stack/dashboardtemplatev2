# Belivert Lovable Dropzone

Plaats hier je export uit Lovable.

## Verplichte bestanden

1. `component.tsx`
2. `preview.html`

## Optioneel

1. `styles.css`
2. `notes.md`

## Richtlijnen

1. Laat imports staan zoals in Lovable export.
2. Voeg geen secrets of API keys toe.
3. Als er externe assets zijn, zet ze in `./assets/` en gebruik relatieve paden.
4. Laat me weten zodra de bestanden erin staan; dan zet ik het om naar de bestaande dashboard-template.
